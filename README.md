## 📌 Descripción del proyecto

Este proyecto consiste en el desarrollo de una aplicación interactiva utilizando Streamlit, donde se aplican conceptos fundamentales de programación en Python en contextos de ingeniería.

La aplicación permite realizar análisis de datos, cálculos técnicos y gestión de información mediante una interfaz amigable.

---

## 🧩 Funcionalidades

### 💰 Ejercicio 1 - Flujo de Caja
Permite registrar ingresos y gastos, mostrando un resumen financiero con indicadores claros.

---

### 🛒 Ejercicio 2 - Registro de Ventas
Se registran productos mediante arrays de NumPy y se visualizan en un DataFrame.

---

### ⚙️ Ejercicio 3 - Indicadores de Mantenimiento
Se calculan indicadores como:
- MTBF
- MTTR
- Disponibilidad

Aplicado a equipos en contexto minero.

---

### 🏗️ Ejercicio 4 - Diseño de Mezcla de Concreto
Uso de programación orientada a objetos (POO) para calcular:
- Volumen de concreto
- Volumen ajustado
- Cantidad de cemento
- Número de sacos

Incluye operaciones CRUD (crear, leer, actualizar y eliminar).

---
